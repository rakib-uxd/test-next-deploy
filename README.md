- Project Management

## Installation

Install ETTMS-V2 with npm

Clone the project

```bash
  git clone https://github.com/Ultra-X-AP/ETTMS-V2
```

Go to the project directory

```bash
  cd ETTMS-V2
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  nodemon
```

## Running Tests

To run tests, run the following command

```bash
  npm run test
```
